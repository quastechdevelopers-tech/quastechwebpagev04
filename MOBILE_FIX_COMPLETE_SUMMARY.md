
**Problem:** Awkward wrapping, inconsistent sizing

```tsx

```

### **6. Form Header** ‚úÖ
**Problem:** Tight spacing, small touch targets  
**Solution:** Responsive padding and icon sizing
```tsx
// Padding: p-4 md:p-5
// Icon:    w-10 h-10 md:w-12 md:h-12
// Text:    text-[10px] md:text-xs
```
